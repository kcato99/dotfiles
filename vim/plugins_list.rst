========================================
入れてるプラグインのリスト
========================================

---------------------------------------------
管理系
---------------------------------------------
dein.vim: plugin manager
- https://github.com/Shougo/dein.vim

---------------------------------------------
補完系
---------------------------------------------
coc.nvim: Intellisense engine for vim8 & neovim, full language server protocol support as VSCode
- https://github.com/neoclide/coc.nvim

---------------------------------------------
表示系
---------------------------------------------
nerdtree: a tree explorer plugin for vim.
- https://github.com/preservim/nerdtree

nerdtree-git-plugin: show git stauts on nerdtree
- https://github.com/Xuyuanp/nerdtree-git-plugin

---------------------------------------------
編集系
---------------------------------------------
cascading: cascading of operator for php/ruby/perl
- https://github.com/c9s/cascading.vim

NERD_commenter.vim: toggle comment
- http://github.com/scrooloose/nerdcommenter

Tabular.vim: arrange sentences in the selected range with specified characters
- https://github.com/godlygeek/tabular

---------------------------------------------
移動系
---------------------------------------------
camelcasemotion: smooth word move of CamelCase and snake_case
- https://github.com/vim-scripts/camelcasemotion

---------------------------------------------
検索系
---------------------------------------------
vim-easymotion: search word cross the screen
- https://github.com/easymotion/vim-easymotion
- http://haya14busa.com/vim-easymotion-ver-3-dot-0-overwin-motions/

eregex.vim: Perl/Ruby style regexp notation for Vim
- https://github.com/othree/eregex.vim

grep.vim: Plugin to integrate Grep search tools with Vim
- https://github.com/yegappan/grep

ctrlpvim/ctrlp.vim : ファイル/Bufferの曖昧検索

バッファ管理系
---------------------------------------------
bufexplorer.vim : <Leader>beでバッファのリストを表示

minibufexpl.vim : タブエディタ風にバッファ管理ウィンドウを表示
- http://www.vim.org/script.php?script_id=159
